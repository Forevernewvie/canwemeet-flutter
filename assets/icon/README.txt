Place icon source PNGs here:
- assets/icon/app_icon.png (1024x1024)
- assets/icon/app_icon_fg.png (foreground for adaptive icon)
- (optional) assets/icon/app_icon_mono.png (monochrome for Android 13)

After updating icon sources, regenerate platform icons:
- dart run flutter_launcher_icons

Play Store icon export (512x512 PNG):
- distribution/store_icon_512.png
