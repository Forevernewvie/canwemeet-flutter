# Play 스토어 리스팅 (한국어)

## 앱 이름
우리 제법 잘 어울려

## Short description (80자 이내)
연인 대화에 바로 쓰는 영어 문장 3개+패턴 3개를 매일 학습해요.

## Full description
영어권 연인과 대화할 때 말문이 막히는 순간, 오늘 바로 써먹을 수 있는 문장과 패턴을 간단하게 학습하세요.

우리 제법 잘 어울려는 매일 핵심 문장 3개와 패턴 3개를 제공해, 영작 부담 없이 실제 대화에 가까운 표현을 익히도록 돕습니다.
상황별로 문장을 고르고, 저장하고, 복습하면서 자연스럽게 말하기 감각을 쌓아보세요.

주요 기능
- 오늘의 학습: 매일 문장 3개 + 패턴 3개 추천
- 상황 탐색: 데이트/갈등/칭찬/미안 등 상황별 문장 찾기
- 문장 상세: 영문/한글 의미 확인, 즐겨찾기 저장
- 내 라이브러리: 저장/복습/통계 탭으로 학습 흐름 관리
- 복습 루프: again/hard/easy 기반 복습으로 반복 학습
- 연속 학습: 스트릭/월간 학습일 확인
- 발음 학습 기록: 학습 이벤트를 누적해 습관 형성
